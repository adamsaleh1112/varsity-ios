### What you can do
- Browse a list of **schools**

Each school shows its **logo** so you can recognize the team at a glance.

### Where the data comes from
Varsity is connected to your Supabase backend, so when the starter data (schools, teams, games) is updated in the database, the app reflects those changes automatically.
